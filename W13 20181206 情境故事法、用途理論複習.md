# 1. 情境故事法
1. 宏觀情境因素分析：從 SET(Social, Economic, Technology) 角度切入觀察
2. 角色界定：瞭解使用者的特性、事件、產品與環境
3. 故事(storyboard)
4. 列出關鍵議題(critical issue)：歸納並討論最重要的 10 個議題，依重要性排列出來

# 2. 用途理論：讓創新可以預期
* 情境的規劃：界定用途的根本
* 進步→用途
* 進步分三層面
    * 功能面
    * 社會面
    * 情感面
1. 顧客為了獲得進步，想獲得哪些體驗？
2. 需要移除那些障礙？
    * 困擾發生在什麼情境？
    * 進步的阻礙是什麼？
    * 如何定義好的品質？
    * 消費者願意做哪些「取捨」來換取更好的品質？
3. 在社會面、情感面與功能面訴求那些用途？
